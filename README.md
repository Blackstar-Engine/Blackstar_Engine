# Blackstar Engine

**Description**

The Blackstar Engine is a utility bot designed for usage in the SCP Site Roleplay faction, **The Blackstar Corporation**. It is designed to carry out tasks, assist with faction managment, and store import data such as points, departments, ranks, and member statistics.

**Folders;**
* /cogs - stores the Blackstar Engines's primary modules and systems
* /utls - contains utilities such as constants and functions that are used throughout the bot
* /ui - contains Discord UI components such as buttons, dropdowns, and interactive elements

**Current Systems & Functions**
* Database System - stores member data such as points, ranks departments, and overall statistics
* Leaderboard System - generates rankings based on stored total points
* Profile System - retrives and displays stored member data
* Enlistment System - enlistments members into departments manually and automatically
* Leave of Absence System - allows members to take LoA's that are stored into the bot's database
